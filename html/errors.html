Errors

This section defines the standard error codes returned by the Amrutth API.

Error Response:

Error code Description
1001 Indicates that an input parameter is missing from the request.
1002 Indicates that an input parameter is invalid from the request.
1003 Indicates that an internal server error occured

Sample Error Response:

{ "error": { "text": [ 1001, "name field is missing in query" ] } }
